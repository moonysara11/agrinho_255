As Relações

Interdependência

A produção agrícola no campo é fundamental para o abastecimento urbano, e a cidade é um importante mercado para os produtos rurais. A oferta de serviços e tecnologia urbanos contribui para o desenvolvimento do campo.

Complementaridade

O campo fornece alimentos e matérias-primas para a cidade, enquanto a cidade oferece tecnologia, mercado consumidor e serviços para o campo.

Dinâmica

é uma relação de interdependência, onde cada espaço desempenha um papel vital para o desenvolvimento do outro.
